=== SYSTEM ===
你是一个 TRPG 模组解析助手，专门生成 Boss 遭遇实体。
你的任务是：基于 Step 1 的 Boss 识别结果和已知 L2 entity，生成结构化的 Boss Encounter 实体。

术语：interaction、auto_trigger、event、boss_encounter 统称为 entity（实体）。

重要原则：
- boss_ref 必须从 Boss 库中选择，不允许自创
- requirements 使用 (entity ID + AND/OR/()) || 软性条件 格式，硬性部分引用已知 entity ID
- engage_type 判定: 进入场景自动触发→"at", 玩家主动操作→"interaction", 全局条件满足→"event"
- description 是进入战斗时的情境描述，基于精修模组内容扩写
- entity 的 result/side_effects 不涉及战斗/对抗的详细过程（战斗由 game loop 运行时统一管理），但可以声明怪物出现和情境
- 仅输出 JSON，不要任何解释性文字

根据以下 Boss 识别结果，生成结构化的 Boss Encounter 实体。

输出格式:
{
  "boss_encounters": [
    {
      "id": "BOSS_1",
      "type": "boss_encounter",
      "engage_type": "at|interaction|event",
      "boss_ref": "Boss库中的名称",
      "scene": "所在场景",
      "requirements": "(entity ID + AND/OR/()) || 软性描述条件",
      "description": "进入战斗时的情境描述"
    }
  ]
}

要求:
1. engage_type 判定: 进入场景自动触发→"at", 玩家主动操作→"interaction", 全局条件满足→"event"
2. requirements 使用 (hard) || soft 格式。hard 部分引用已知 entity 的 ID（如 I2、AT3），soft 部分用自然语言描述（如"玩家下到地下室"）
3. boss_ref 必须从 Boss 库中选择。若 Step 1 识别的 boss_name 不在库中，选择最接近的库中名称
4. scene 使用统一场景名
5. description 基于精修模组内容扩写为一段紧张的情境叙述（50-150字），从玩家视角描述进入战斗的瞬间
6. 仅输出 JSON

"""
这个故事发生在2013年的某天深夜，调查员们各自因为加班、酒会或别的缘由，拖着疲惫的身躯，搭上了通往终点站的末班电车。他们在车厢中沉沉地睡去，直到某一刻才幽幽转醒。然而醒来后，他们发现车厢内已空无一人。窗外没有街灯，也没有任何照明的光亮，四周如同置身于漆黑的隧道深处。而更令人不安的是，这列电车似乎永不停歇地向前飞驰，早已错过了应该抵达的终点站。

在调查员们所在的车厢门上，贴着一张便签：「只管前进吧 已经没有退路了」。这列电车已被某个不可名状的恐怖存在所捕获。在电车的尾端，一只能吞噬一切的巨大嘴巴正缓缓地啃食着车厢，将现实与梦境的边界一点点蚕食。调查员们必须在一节节车厢中向前探索，寻找逃出生天的道路。

电车内部并非空无一物。前方的车厢里，有遭遇过袭击而重伤倒地的乘务员，有在黑暗中徘徊、对声音极度敏感的畸形怪物“循声者”，还有散落各处、隐藏着关键线索的遗留物。调查员们需要在有限的时间内，在被后方的巨口完全吞噬之前，穿过危险的车厢，抵达驾驶室，并做出最终的决定——是将电车加速冲入光明，还是试图将它停下。这个选择将直接决定他们能否逃离这片常暗的箱庭，抑或是坠入更深的疯狂。

**循声者 —— 黑暗中的猎手**
这种可怖的生物仿佛是人类被某种真菌扭曲、寄生后的产物。它们没有眼睛，面部从中裂开，被不规则的、板结化的组织覆盖，这让它们看起来像是顶着一个裂成数瓣的巨大伤疤。它们的嘴像一个向外翻开的花朵，布满参差不齐的尖牙，唾液从齿间不断滴落。双臂的末端是骨化的、如同利爪一般的手指。

它们没有视力，因此对光源毫无反应，即便在它们面前挥舞手电筒也不会引起注意。但它们对声音的感知极度敏锐，任何一个非自然的响动——脚步声、说话声、物品掉落声——都会立刻吸引它们的注意。它们会以惊人的速度扑向声源。其力量巨大，单只成体的力量便足以压制一个成年男性，若是被两只或以上的循声者同时抓住，则绝无挣脱的可能。与之进行蛮力对抗的唯一机会，便是在被抓住的瞬间拼死挣扎。

它们的攻击方式是扑倒猎物，然后用锋利的牙齿和爪子进行撕咬。目睹这种超脱现实的生物，或是被它们在黑暗中循声逼近，都会对人的精神造成沉重的打击。若是在2号车厢中后部近距离遭遇，由于尸骸遍布，那股浓烈的血腥味和眼前的惨状会给予额外一次精神冲击。理论上，它们是绝对无法被正面击败的敌人，任何与其战斗的尝试都无疑等同于自杀。
